Release summary

This update records the Gumroad release of the English Early Access Edition of AI of One’s Own.

This Early Access release includes the English eBook and practical materials for building a personal AI environment. The project focuses on understanding the structure behind AI tools: Docker, MCP, RAG, memory, workspace design, document handling, Google integration, SQLite, and backup workflows.

What this release means

This is the first public Gumroad release of the English edition. It marks the transition from an internal project into a public product with an official website, support site, and publication record.
