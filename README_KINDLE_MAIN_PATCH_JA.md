# Kindle main patch

## 目的

英語版・日本語版・.com直下トップページで、Kindleをメイン導線に変更します。

## 修正内容

- `.com` 直下 `index.html`
  - Kindleをメイン導線に変更
  - English Kindle / Japanese Kindle のボタンを主表示
  - GumroadはTemplate Pack版としてサブ表示
  - 著者写真を少し大きめの四角い写真へ変更

- `/en/index.html`
  - Kindleをメイン導線に変更
  - GumroadはTemplate Pack editionとして下部に移動

- `/ja/index.html`
  - Kindleをメイン導線に変更
  - Gumroadはテンプレートパック版として下部に移動
